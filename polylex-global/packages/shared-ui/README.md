# @polylex/shared-ui

Shared React UI primitives and hooks for PolyLex apps (frontend + Zalo mini app).

## Included

- Components: `SocialLoginButton`, `ErrorToast`, `LoadingSpinner`, `AuthGuard`
- Hooks: `useSocialLogin`, `useAuthToken`, `useApi`
- Types: social-login and UI primitive types

## Build

```bash
npm run build --workspace=packages/shared-ui
```

## Test

```bash
npm run test --workspace=packages/shared-ui
```
